# Image-Slider

# 🎞️ All-in-One Image Slider

## 📌 Project Description
This project is a fully functional and responsive **Image Slider (Carousel)** built using **HTML, CSS, and JavaScript**.

It allows users to navigate through multiple images using:
- Previous/Next buttons
- Navigation dots
- Automatic slideshow (auto-play)

The slider also includes smooth animations, captions, and responsive design for different screen sizes.

---

## 🚀 Features
- ✅ Automatic image sliding (every 4 seconds)
- ✅ Manual navigation (Next/Previous buttons)
- ✅ Dot navigation for direct slide selection
- ✅ Smooth fade animation effect
- ✅ Image captions for better description
- ✅ Responsive design (mobile-friendly)
- ✅ Looping slideshow (continuous cycling)

---

## 🛠️ Technologies Used
- **HTML5** – Structure of the slider  
- **CSS3** – Styling, animations, responsiveness  
- **JavaScript (Vanilla JS)** – Logic and interactivity

## 🎮 How It Works
- The slider starts automatically when the page loads
- Images change every **4 seconds**
- Users can:
  - Click **Next/Prev buttons** to navigate
  - Click **dots** to jump to a specific slide
- Timer resets when user interacts

---

## 🧠 Core Logic
- `showSlides(n)` → Displays current slide and hides others  
- `moveSlide(n)` → Moves forward/backward  
- `jumpToSlide(n)` → Direct navigation  
- `autoPlay()` → Runs slideshow automatically  
- `resetTimer()` → Prevents timer conflicts
